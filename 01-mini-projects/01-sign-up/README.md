# Sign-Up Form

A simple **Sign-Up Form** built using **HTML**, **CSS**, and **JavaScript**.  
It demonstrates form validation, accessibility, and basic JS interactivity.

---

## Features

- Full Name, Email, Password  
- Password confirmation validation  
- Autofill-friendly fields using `autocomplete`  
- Success message on valid submission  
- Form reset after success  
- Dark theme design

---

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)

---

## Folder Structure

SignUpForm/
│
├── index.html
├── style.css
├── script.js
└── README.md

---

## How to Run

1. Open `index.html` in a browser.  
2. Fill in all required fields.  
3. Submit to see the success message.
